Achsenausrichtung am Smartphone
X
X – Links / Rechts (horizontal)
Positiv = rechts (Handy aufrecht, Portrait)
Y
Y – Oben / Unten (vertikal)
Positiv = Richtung Oberkante des Handys
Z
Z – Senkrecht zum Display
Positiv = aus dem Display heraus. Flach liegend = Auf/Ab-Richtung!
Σ
Total – √(X²+Y²+Z²)
Kombination aller Achsen. Basis für DIN 4150-2 Auswertung.
Messtipps

Flach auf Oberfläche: Z-Achse = vertikale Schwingung (Auf/Ab)

Aufrecht anlehnen: Y-Achse = vertikale Schwingung

Feste Kopplung! Handy direkt auflegen, nicht freihalten.

RMS: mind. 10–30 s messen für aussagekräftige Werte.

Verschiebung (mm): doppelte Integration – nur Orientierung, kann driften.

Nicht kalibriert! Für normgerechte Nachweise ist ein kalibriertes Gerät erforderlich.

Als App installieren (PWA)

1. Seite in Chrome öffnen

2. Menü (⋮) antippen

3. „App installieren" oder „Zum Startbildschirm hinzufügen"

4. „Installieren" bestätigen

✅ App erscheint am Homescreen – ohne App Store!

Tipp: Oben erscheint ggf. automatisch ein Install-Banner.

1. Seite in Safari öffnen (nicht Chrome!)

2. Teilen-Symbol (□↑) antippen

3. „Zum Home-Bildschirm" wählen

4. Name bestätigen → „Hinzufügen"

✅ App erscheint am iPhone-Homescreen!

Beim ersten Start: Sensor-Erlaubnis erteilen.

Was ist eine PWA?

Progressive Web App = sieht aus und fühlt sich an wie eine App:

✅ Icon am Homescreen

✅ Startet im Vollbild

✅ Kostenlos – kein App Store

✅ Offline-fähig (nach erstem Laden)

✅ Updates automatisch über GitHub Pages
